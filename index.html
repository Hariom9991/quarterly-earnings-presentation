def calculate_operating_margin(revenue, operating_expenses):

Risk Assessment Query

Credit Risk Analysis using SQL Analytics 


-- Calculate portfolio risk metrics
SELECT 
    portfolio_segment,
    COUNT(DISTINCT account_id) AS total_accounts,
    SUM(loan_amount) AS total_exposure,
    AVG(credit_score) AS avg_credit_score,
    SUM(CASE WHEN days_past_due > 90 THEN 1 ELSE 0 END) 
        AS high_risk_accounts,
    ROUND(AVG(probability_of_default) * 100, 2) 
        AS avg_default_prob_pct
FROM credit_portfolio
WHERE reporting_date = '2024-12-31'
GROUP BY portfolio_segment
HAVING total_exposure > 1000000
ORDER BY avg_default_prob_pct DESC;
Result: Portfolio default probability decreased to 1.3% from 1.7% in Q3, reflecting improved risk management and credit quality
This SQL query demonstrates our credit risk assessment methodology. We analyze our loan portfolio across segments, calculating key risk indicators including default probabilities and delinquency rates. The improved metrics show effective risk management.

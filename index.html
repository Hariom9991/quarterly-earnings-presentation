Data Pipeline (Code Example)


import pandas as pd

# Load transactional data
df = pd.read_csv("transactions.csv")

# Aggregate revenue by quarter
quarterly = (
    df.groupby("quarter", as_index=False)["revenue"]
      .sum()
      .rename(columns={"revenue": "total_revenue"})
)

# Compute year-over-year growth
quarterly["yoy_growth"] = (
    quarterly["total_revenue"]
      .pct_change(periods=4)
      .round(4)
)

print(quarterly)
This example illustrates how we generate the quarterly aggregates and year-over-year growth rates directly from raw transactional data using Python and pandas. In practice, this runs inside our analytics pipeline.

Financial Projection Formula

Compound growth of quarterly revenue:

\[ R_t = R_0 \left(1 + \frac{g}{n}\right)^{n \cdot t} \]

Where:

  • \( R_t \): projected revenue at time \( t \)
  • \( R_0 \): current revenue base
  • \( g \): expected annual growth rate
  • \( n \): number of compounding periods per year
